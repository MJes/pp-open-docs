| -------------- | ------------ | ---- | ------------------------------- | --------------------------------- |
| jnStoreId        | Long       | 是   | 1234                             | 巨牛店铺id                         |
| jnGoodsId        | String(32) | 是   | 1234                             | 巨牛商品id                         |
| jnSpecList       | Spec[]     | 是   |                                  | 巨牛颜色尺码列表                    |

| 字段            | 类型         |是否必须| 示例                            | 描述                              | 
| -------------- | ------------ | ---- | ------------------------------- | --------------------------------- |
| jnId           | Long         | 是   |  1234                         | 巨牛规格id                         |
| jnStock        | Integer      | 是   | 68                            | 库存                              |
