# 请求方式
post  



# 请求url
/goods/open/updateStock



# 请求
   字段  |  类型  |  是否必须  |  示例  |  描述  
------------------------------|---------------|------|-----------------------------------|-----------------------------------
goodsId                |  Long     |  是  | 123 | 商品id
specId                 |  Long     |  是  | 12 | 规格id
stock                  |  Integer     |  是  |  99                    |  库存数量



# 响应
## 整体
   字段  |  类型  |  示例  |  描述  
------------------------------|---------------|-----------------------------------|-----------------------------------
status                        |  String      |  处理结果                           |  处理结果返回值 UPDATE_STOCK_SUCCESS 成功 UPDATE_STOCK_FAIL 成功
